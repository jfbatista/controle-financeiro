## Controle Financeiro para Pequenos Negócios

Aplicação web + PWA para controle financeiro de pequenos comércios pensada
como sistema moderno, multi-dispositivos e escalável.

### Objetivo

- Centralizar entradas e saídas de caixa.
- Mostrar lucro/prejuízo de forma simples e visual.
- Ajudar o dono do negócio a tomar decisões rápidas, sem precisar
  saber mexer em Excel.

### Stack Principal

- **Frontend**: React + TypeScript + Vite + Chakra UI (PWA).
- **Backend**: Node.js + NestJS + Prisma.
- **Banco de Dados**: MySQL.
- **Autenticação**: JWT (access + refresh).

### Estrutura do Repositório

- `frontend/` – código do app web/PWA.
- `backend/` – API em NestJS + Prisma.
- `docs/` – documentação funcional e técnica.

### Como rodar (visão rápida)

- Veja `backend/README.md` para subir a API (NestJS + Prisma).
- Veja `frontend/README.md` para subir o app web (React + Vite).
- Veja `docs/` para visão geral, arquitetura, regras de negócio e deploy.
